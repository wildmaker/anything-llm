## 文档使用指南

- **项目启动**: 先阅读需求类文档，了解产品目标和核心功能
- **技术开发**: 参考设计类和技术类文档，掌握系统架构和实现细节
- **部署实施**: 按照技术类文档中的实施计划进行系统部署或开发环境搭建
- **协作开发**: 遵循方法论类文档中的开发方法论，实践TDD和敏捷开发
- **问题排查**: 利用工具类文档解决常见问题和系统维护
